# Assignments & rater matrix
Status: Draft (2026-03-03)

## Scope
Матрица назначений определяет “кто кого оценивает” в рамках кампании.
HR назначает вручную; система умеет автопредлагать назначения.

## Autogeneration (agreed)
MVP: HR выбирает подразделения (узлы дерева). Система включает сотрудников выбранных подразделений и всех дочерних.

Назначения предлагаются на базе direct-manager отношений (из снапшота):
- `manager`: прямой руководитель subject (если участвует).
- `subordinates`: прямые подчинённые subject.
- `peers`: сотрудники с тем же прямым руководителем.
  - Это покрывает правило: “руководители подразделений одного уровня — коллеги, подчиняются вышестоящему”.
- `self`: самооценка (default on, configurable).

## Editing & lock
- До `campaign.locked_at` матрица может редактироваться HR.
- После `campaign.locked_at` (первый draft save): матрицу менять нельзя (зафиксировано).

## Roles per rater
Один rater может иметь разные роли для разных subject (зафиксировано).
В рамках одной пары (rater, subject) в MVP предполагаем одну роль (без усложнений).
