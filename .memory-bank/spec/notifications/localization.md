# Localization & template versioning (MVP: RU-only)
Status: Draft (2026-03-03)

Зафиксировано:
- MVP язык уведомлений: **RU-only**.
- Архитектурно закладываем возможность добавить другие языки (например, EN), но не реализуем в MVP.
- Шаблоны **версионируются**.

## Approach (SSoT rules)
- У каждого уведомления есть:
  - `template_key` (например, `campaign_invite`)
  - `template_version` (например, `v1`)
  - `language` (в MVP всегда `ru`)
- В будущем `language` выбирается по настройке компании/сотрудника, но MVP фиксирует `ru`.

Связанные документы (аннотированные ссылки):
- [Templates RU v1](templates-ru-v1.md): ключи шаблонов и переменные. Читать, чтобы добавлять/менять шаблоны без поломки совместимости.
