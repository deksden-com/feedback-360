# Data retention & privacy (MVP)
Status: Draft (2026-03-03)

## Visibility
- Employee/Manager: никогда не видят raw-комментарии других людей; видят только агрегаты и AI-processed текст при выполнении threshold.
- HR Admin/HR Reader: видят raw-комментарии (MVP решение).

## Retention
MVP: явная политика сроков хранения не задана (фиксируем как риск). До уточнения считаем данные хранящимися до удаления/архивации кампании по политике компании.
