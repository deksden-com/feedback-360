# C4 L1 — System Context
Status: Draft (2026-03-03)

## System
**feedback-360** — внутренняя корпоративная система для проведения оценки сотрудников методом “360 градусов”, с best practices: анонимность, группы оценщиков, веса, заморозка правил после старта/первого ответа, аудитность.

## People (actors)
- **HR Admin**: настраивает справочник, оргструктуру, модели компетенций, кампании, назначения; видит прогресс и результаты; управляет AI-обработкой.
- **HR Reader**: читает прогресс/результаты в read-only режиме, но без доступа к raw-комментариям.
- **Manager**: видит результаты по своей команде в рамках правил анонимности.
- **Employee**: заполняет анкеты и видит свой дашборд результатов (без raw комментариев других людей).

## External systems
- **Supabase Auth**: аутентификация (magic link email в MVP).
- **Supabase Postgres**: основная БД.
- **Resend**: транзакционные email.
- **AI Processing Service (external)**: постобработка текстовых комментариев; отвечает webhook’ом.
- **Vercel (hosting + cron)**: хостинг web и планировщик задач (напоминания/окончание кампаний/outbox dispatch).
