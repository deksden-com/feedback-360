# Assets Index
Status: Draft (2026-03-06)

Этот раздел хранит **не-нормативные артефакты** проекта: визуальные референсы, скриншоты, вспомогательные материалы. SSoT поведения системы остаётся в `spec/`, а evidence реализации — в `.memory-bank/evidence/`.

- [UI assets](ui/index.md): библиотека визуальных референсов для web UI, включая распакованный `stitch_go360go`. Читать, чтобы понимать, какие макеты можно использовать как вдохновение и где лежат исходные файлы.
- [Visual references policy](../mbb/visual-references.md): правила хранения и использования визуальных референсов, включая provenance и запрет трактовать макеты как бизнес-спецификацию. Читать перед добавлением новых дизайн-источников, чтобы не засорять меморибанк и не создавать конфликт с SSoT.
