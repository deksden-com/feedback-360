# Memory Bank — Index

- [Memory Bank Bible (MBB)](mbb/index.md): Правила ведения меморибанка (SSoT, атомарность, индексы, аннотированные ссылки, duo pattern). Читать перед добавлением/изменением документов, чтобы не плодить дубли и не ломать навигацию.
- [Specifications](spec/index.md): SSoT требований (WHAT): доменные правила, C4, ERD, RBAC, state machines, расчёты, уведомления, webhook security, CLI, UI sitemap. Читать, чтобы реализовывать поведение системы и писать тесты по спецификации.
- [Plans](plans/index.md): SSoT плана работ: эпики/фичи/vertical slices и сценарии приёмки. Читать, чтобы двигаться в правильном порядке и закрывать definition of done фич через автосценарии.
- [ADR](adr/index.md): SSoT решений (WHY): ключевые компромиссы и мотивация. Читать, чтобы понимать “почему так” и не отменять решения неосознанно.
- [Project structure](structure.md): Структура репозитория и границы частей проекта (apps/packages/docs), включая слои и места для vertical slices. Читать, чтобы правильно раскладывать код и избегать утечки бизнес-логики в UI/CLI.
