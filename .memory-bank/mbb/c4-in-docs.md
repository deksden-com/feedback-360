# MBB — C4 in docs
Status: Draft (2026-03-03)

Мы используем C4 как “каркас” документации:
- **L1**: система целиком (контекст, акторы, внешние интеграции).
- **L2**: контейнеры (web/core/db/auth/email/ai/cron).
- **L3**: компоненты внутри контейнеров (use-cases/policies/outbox/webhook handler).

Где это отражено (аннотированные ссылки):
- [C4 Index](../spec/c4/index.md): список L1/L2/L3 документов проекта. Читать, чтобы видеть архитектуру целиком и не ломать границы ответственности.
