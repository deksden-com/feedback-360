# MBB Templates — Index
Status: Draft (2026-03-03)

- [Epic template](epic.md): шаблон описания эпика (цель, scope, фичи, зависимости, риски, definition of done). Использовать при заведении нового эпика в планах.
- [Feature template](feature.md): шаблон описания фичи как вертикального слайса (value→deliverables→scenarios→tests→seeds). Использовать, чтобы каждая фича имела автопроверяемый сценарий и не размывалась.
